Cuentas de usuario

Registro de usuarios#

Para poder acceder a nuestros servicios en línea tiene que crear una cuenta en nuestro sistema. Para hacerlo siga los pasos a continuación.

Haga click aquí, para acceder al formulario de creación de usuarios.

De forma alternativa, puede encontrar el formulario de registro desde el menú principal de nuestro sitio web:

Se desplegará este formulario:

Para poder crear una cuenta en nuestro sistema debe seguir las siguientes reglas al completar el formulario:

  1. Nombre: Ingrese su nombre completo
  2. Usuario: Hasta 15 caracteres. *
  3. Dirección de correo electrónico: Correo electrónico válido. Los correos con dominios de Yahoo no están permitidos. *
  4. Contraseña: Mínimo 8 caracteres. Puede contener números, letras y caracteres especiales.

* Este valor debe ser único en nuestro sistema, así que si ya otra persona usó el nombre de usuario o email ingresado, el sistema no permitirá el registro y tendrá que probar con otro diferente.

Correo ya existe

En caso de que ya exista un correo con su registro, intente recuperar la cuenta de usuario que ya tenía registrada siguiendo los pasos que se muestran aquí.

Nota:

El valor que use en el campo Usuario será el valor que tiene que usar para iniciar sesión en el formulario de ingreso.

Después de completar el formulario y presionar el botón Registrar:

  • Si hay algún error, el sistema le notificará.
  • Si todo está bien, el sistema le enviará un correo con sus datos para ingresar a las Consultas Registrales.

Inicio de sesión#

Haga click aquí, para acceder al formulario de inicio de sesión.

De forma alternativa, puede encontrar el formulario de inicio de sesión a nuestros servicios web, desde el menú principal de nuestro sitio web:

En caso de que no esté autenticado en nuestro sistema, se desplegará el siguiente formulario para iniciar sesión:

¿No tiene cuenta para acceder a nuestro sistema?

Siga los pasos de esta guía para obtener una cuenta.

En caso de que haya olvidado su contraseña, puede crear una nueva siguiendo estos pasos.

Cambio de contraseña#

  1. Haga click aquí para acceder al formulario de cambio de contraseña.
  2. Ingrese la dirección de correo electrónico con la que se registró en nuestro sistema.
  3. Presione Enviar.
  4. Se le enviará un código de verificación. Una vez que lo haya recibido, podrá seleccionar una nueva contraseña para su cuenta.
