## Registro de usuarios

Para poder acceder a nuestros servicios en línea tiene que crear una cuenta en nuestro sistema. Para hacerlo siga los pasos a continuación.

Haga <a href="https://registro-publico.gob.pa/index.php?option=com_content&view=article&id=809&Itemid=741" target="_blank">click aquí</a>, para acceder al formulario de creación de usuarios.

De forma alternativa, puede encontrar el formulario de registro desde el menú principal de nuestro sitio web:

<img alt="Registro de usuarios" src={useBaseUrl('img/servicios-web-registro-1.png')} />

Se desplegará este formulario:

<img alt="Formulario de registro" src={useBaseUrl('img/servicios-web-registro-2.png')} />

Para poder crear una cuenta en nuestro sistema debe seguir las siguientes reglas al completar el formulario:

1. **Nombre**: Ingrese su nombre completo
1. **Usuario**: Hasta 15 caracteres. *
1. **Dirección de correo electrónico**: Correo electrónico válido. Los correos con dominios de Yahoo no están permitidos. *
1. **Contraseña**: Mínimo 8 caracteres. Puede contener números, letras y caracteres especiales.

\* Este valor debe ser único en nuestro sistema, así que si ya otra persona usó el nombre de usuario o email ingresado, el sistema no permitirá el registro y tendrá que probar con otro diferente.

:::tip Correo ya existe
En caso de que ya exista un correo con su registro, intente recuperar la cuenta de usuario que ya tenía registrada siguiendo los pasos que se muestran [aquí](#cambio-de-contraseña).
:::

:::note Nota:
El valor que use en el campo **Usuario** será el valor que tiene que usar para iniciar sesión en el formulario de ingreso.
:::

Después de completar el formulario y presionar el botón **Registrar**:
  - Si hay algún error, el sistema le notificará.
  - Si todo está bien, el sistema le enviará un correo con sus datos para ingresar a las Consultas Registrales.

## Cambio de contraseña

1. Haga <a href="https://registro-publico.gob.pa/index.php?option=com_users&view=reset&Itemid=743" target="_blank">click aquí</a> para acceder al formulario de cambio de contraseña.
1. Ingrese la dirección de correo electrónico con la que se registró en nuestro sistema.
1. Presione **Enviar**.
1. Se le enviará un código de verificación. Una vez que lo haya recibido, podrá seleccionar una nueva contraseña para su cuenta.
