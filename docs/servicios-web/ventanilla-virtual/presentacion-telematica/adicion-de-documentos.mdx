## Acceso al formulario

Acceder al formulario para Adición de Documentos mediante presetanción telemática, siguiendo estos pasos:

<img alt="Acceso a formulario de adición de documentos" src={useBaseUrl('img/servicios-web-ventanilla-virtual-telematica-adicion-1.jpg')} />

## Liquidacion

### Solicitante
Complete esta sección de la misma forma como se explica en la <a href="./presentacion-de-documentos#solicitante" target="_blank">Guía para Presentación Telemática de documentos</a>.

### Trámites
Complete esta sección de la misma forma como se explica en la <a href="./presentacion-de-documentos#tramites" target="_blank">Guía para Presentación Telemática de documentos</a>.

:::info Si el defecto de su escritura fue: Cobro adicional
Llenar el formulario con la siguiente información:
- **Tipo de Trámite**: Registro
- **Descripción**: Cobro adicional
- **Valor Base**: Ingresar el monto a pagar que se le indicó en el defecto de su entrada
:::

### Documento adjunto

:::info Si está adicionando documentos que no sean escrituras públicas
Deje el formulario en blanco
:::

Complete esta sección de la misma forma como se explica en la <a href="./presentacion-de-documentos#documento-adjunto" target="_blank">Guía para Presentación Telemática de documentos</a>.

**Tenga en cuenta lo siguiente:**

### Dueño del Documento
Complete esta sección de la misma forma como se explica en la <a href="./presentacion-de-documentos#dueño-del-documento" target="_blank">Guía para Presentación Telemática de documentos</a>.

### Entrada Original
Con este formulario le indica al sistema la identificación de la entrada original a la que se le realizará la adición:
><img alt="Formulario de selección de entrada original" src={useBaseUrl('img/servicios-web-ventanilla-virtual-telematica-adicion-2.jpg')} />
>
> - **BUSCADOR**: Al presionar este botón se desplegará un subformulario para que ingrese el **Año** y **Número** de la entrada original.
>
> - **QUITAR ENTRADA ORIGINAL**: Le permite eliminar los datos de la entrada original, en caso de haberse equivocado al ingresar los datos en el **BUSCADOR**.

## Guardar presentación
El botón <img alt="GUARDAR" src={useBaseUrl('img/btn-guardar.png')} /> le permite guardar todos los datos introducidos hasta el momento, ya sea para no perder la información o para realizar el pago posteriormente.

Consulte la sección <a href="./informacion-adicional#recuperar-presentaciones-guardadas" target="_blank">Recuperar presentaciones guardadas</a> para más información sobre este tema.

## Realizar pago
Cuando esté listo para realizar el pago puede presionar el bogón <img alt="PAGAR/SOLICITAR" src={useBaseUrl('img/btn-pagar-solicitar.png')} />. A continuación se desplegará el siguiente formulario:

> <img alt="Confirmación de pago" src={useBaseUrl('img/servicios-web-ventanilla-virtual-telematica-7.png')} />
>
> Al presionar el botón **REALIZAR PAGO**, se le desplegará la pasarela del banco para completar el pago con tarjetas de crédito _VISA_ O _MASTERCARD_.
>
> Si presiona el botón **CERRAR** regresará al listado de presentaciones guardadas desde donde podrá modificar o completar la presentación cuando lo desee. Consulte la sección <a href="./informacion-adicional#recuperar-presentaciones-guardadas" target="_blank">Recuperar presentaciones guardadas</a> para más información.

Al completar la transacción con la pasarela de pago del banco, el sistema le mostrará la pantalla que se muestra a continuación, desde donde puede descargar manualmente su Recibo de Liquidación. Este paso es opcional, porque el sistema automáticamente le envía el Recibo de Liquidación al correo electrónico suministrado en [este paso](#solicitante).

<img alt="Pago aceptado" src={useBaseUrl('img/servicios-web-ventanilla-virtual-telematica-8.jpg')} />

:::caution Todavía no ha terminado
Para completar su presentación telemática, presione el botón **PRESENTAR**. Ver la siguiente sección.
:::

En caso de que presione el botón **VOLVER**, el sistema guardará su solicitud de presentación para que pueda completar la presentación en otro momento. Consulte la sección <a href="./informacion-adicional#recuperar-presentaciones-guardadas" target="_blank">Recuperar presentaciones guardadas</a> para más información.

## Solicitar presentación

:::caution Antes de presionar el botón **Presentar**
Recuerde que solo puede presentar documentos en el Registro Público de Panamá dentro del horario de atención al público: <a href="../../../dincrece/informacion-general#horario" target="_blank">Ver horarios</a>.

Si intenta hacer la presentación fuera del horario, el sistema le mostrará una advertencia con esta información y no lo dejará continuar.
:::

Al presionar el botón **PRESENTAR** en el paso anterior, el sistema procesará su solicitud y en caso de que todo haya salido bien, se desplegarán los datos de su presentación como se muestra a continuación:

<img alt="Presentación de adición completa" src={useBaseUrl('img/servicios-web-ventanilla-virtual-telematica-adicion-3.jpg')} />

:::caution
En la sección **Documentos** haga click sobre el enlace **Justificante** para descargarlo, ya que este documento no se le envía por correo electrónico.
:::

En este punto ya se ha ingresado la adición.
